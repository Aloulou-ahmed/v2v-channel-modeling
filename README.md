Projet V2V — Modélisation du canal (V2V)
=====================================

Résumé
------
Ce dépôt contient des scripts MATLAB utilisés pour l'étude et la simulation du canal véhicule-à-véhicule (V2V).

Structure
---------
- `Constants.txt` : constantes utilisées par les scripts.
- `Code/` : scripts MATLAB :
  - `phase2_cdf_data.m`
  - `phase3_cdf_fitting.m`
  - `phase4_channel_simulation.m`
  - `phase5_bpsk_performance.m`
- `Figures/` : figures générées par les scripts.
- `V2V_Report.pdf` : rapport (si présent).

Utilisation
-----------
1. Ouvrir MATLAB.
2. Placer le dossier du projet dans le `path` MATLAB ou se positionner dans le dossier racine.
3. Lancer les scripts dans l'ordre souhaité (par exemple `phase2_cdf_data.m` puis `phase3_cdf_fitting.m`, ...).
